데이터 출처 안내

한눈체크는 구매 전 확인을 돕기 위해 공공데이터포털을 통해 제공되는 공개 API를 활용합니다. 사이트 내 결과는 각 제공기관의 데이터 반영 시점과 API 응답 상태에 따라 달라질 수 있습니다.

활용 데이터

  • 국세청 사업자등록정보 진위확인 및 상태조회 서비스 — 사업자등록 상태, 과세유형, 휴업·폐업 여부, 진위확인 결과 확인에 사용합니다.
  • 공정거래위원회 통신판매사업자 등록상세 제공 서비스 — 통신판매업 신고정보, 신고번호, 상호명, 대표자명, 사업장 소재지 등 공개 등록정보 확인에 사용합니다.
  • 공공데이터포털 — 위 API의 활용신청, 인증키 발급, 서비스 호출 경로를 제공합니다.

공식 기관 사이트가 아닙니다

본 사이트는 국세청, 공정거래위원회, 공공데이터포털의 공식 서비스가 아니며, 각 기관과 제휴 또는 대행 관계에 있지 않습니다. 정확한 최종 확인이 필요한 경우 각 기관의 공식 조회 화면을 함께 확인해 주세요.

데이터 한계

조회 결과는 각 제공기관의 데이터 반영 시점, 신규 등록·변경 반영 지연, 사용자의 입력값, API 응답 상태에 따라 실제 상태와 차이가 있을 수 있습니다. 한눈체크는 데이터 표시를 돕는 도구이며 거래 결과를 보장하지 않습니다.

브라우저 기반 기기 정보

컴퓨터 사양 확인 기능은 별도 외부 데이터 제공기관을 사용하지 않고, 사용자의 브라우저에서 제공하는 Web API와 WebGL 정보를 바탕으로 운영체제 추정, CPU 논리 코어 수, 메모리 추정값, GPU/WebGL 정보, 화면 해상도 등을 표시합니다.

이 정보는 브라우저와 운영체제의 보안 정책에 따라 제공 범위가 제한될 수 있으며, 정확한 부품명은 사용자가 직접 확인해 입력해야 합니다.

한눈체크

확인 가능한 정보를 한눈에 정리하고 의미를 설명하는 체크 도구 모음입니다. 조회·측정 결과는 참고용이며, 실제 판단을 보장하지 않습니다.
