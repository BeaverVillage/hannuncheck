    Before payment

    입금 전 확인 항목

    • 사이트 하단의 사업자정보와 조회 결과를 비교하세요.
    • 무통장입금만 요구하는 경우 결제수단을 다시 확인하세요.
    • 가격이 비정상적으로 낮으면 환불정책과 고객센터를 확인하세요.
    • 등록정보가 정상이어도 거래 안전을 보장하지 않는다는 점을 기억하세요.

    Safe position

    한눈체크는 사기 판정 서비스가 아닙니다

    계좌번호, 전화번호, 사용자 신고 이력, 피해사례 DB를 다루지 않습니다. 공식 등록정보 확인과 사용자가 직접 비교할 수 있는 체크리스트 제공에 집중합니다.

    Notice

    거래 안전을 보장하지 않습니다

    사업자등록 상태나 통신판매업 정보가 정상으로 확인되어도 실제 거래 안전성을 보장하지 않습니다. 결제수단, 환불정책, 배송정책, 후기, 사이트 하단 정보를 함께 확인해야 합니다.

    Service structure

    한눈체크 기능 구성

    공식 API 조회 결과와 사용자가 입력한 쇼핑몰 정보를 한 화면에서 비교할 수 있도록 구성했습니다.

    01

    사업자등록 상태 확인

    사업자번호 기준으로 계속사업자, 휴업자, 폐업자 여부와 과세유형을 표시합니다.

    02

    통신판매업 정보 확인

    공정거래위원회 등록상세 API를 통해 신고번호, 신고일자, 업소상태 정보를 확인합니다.

    03

    쇼핑몰 정보 비교

    상호명, 대표자명, 쇼핑몰 도메인이 조회 정보와 어느 정도 일치하는지 참고용으로 보여줍니다.

    How to use

    조회된 정보와 쇼핑몰에 표시된 정보가 일치하는지 직접 비교합니다.

    Guide contents

    가이드 콘텐츠

    조회 결과를 오해하지 않도록 사업자등록, 통신판매업, 거래 전 확인 항목을 함께 설명합니다.

    사업자등록번호로 확인할 수 있는 것

    계속사업자, 휴업자, 폐업자 상태와 과세유형의 의미를 정리합니다.

    통신판매업 신고 확인 방법

    온라인 쇼핑몰에서 통신판매업 신고 정보가 왜 중요한지 설명합니다.

    입금 전 확인해야 할 정보

    사업자 정보, 결제수단, 환불정책, 고객센터 확인 항목을 정리합니다.

    FAQ

    자주 묻는 질문

    사업자 정보가 정상이면 안전한 쇼핑몰인가요?

    아닙니다. 정상 등록정보는 참고 자료일 뿐이며, 실제 거래 안전성을 보장하지 않습니다.

    한눈체크에서 사기 여부를 알려주나요?

    사기 여부를 판정하지 않습니다. 공식 등록정보 확인과 비교 체크리스트 제공에 집중합니다.

    계좌번호나 전화번호도 조회하나요?

    조회하지 않습니다. 개인정보와 분쟁 리스크를 줄이기 위해 사업자 정보 확인 중심으로 운영합니다.

    입력한 사업자번호를 저장하나요?

    조회 기능은 Cloudflare Functions에서 즉시 처리하며, 별도 DB에 조회 이력을 저장하지 않는 구조입니다.

    한눈체크

    공식 등록정보를 바탕으로 구매 전 확인 흐름을 돕는 사업자 정보 확인 도구입니다. 사기 여부 또는 거래 안전성을 보장하지 않습니다.
