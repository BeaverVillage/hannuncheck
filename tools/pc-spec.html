홈/컴퓨터 사양 확인

Device check

컴퓨터 사양을 한눈에 확인하세요

브라우저에서 확인 가능한 기기 정보를 자동으로 정리하고, 확인되지 않는 정확한 부품명은 사용자가 직접 입력해 의미를 함께 볼 수 있습니다.

내 컴퓨터 사양 확인

안내: 웹페이지는 보안 정책상 정확한 CPU 모델명, 전체 RAM 용량, 저장장치 모델명, 메인보드 정보 등을 직접 읽을 수 없습니다. 자동 확인값은 참고용이며, 부족한 항목은 아래 직접 입력란에 채워서 해석하면 됩니다.

CPU quick test

CPU 간단 테스트 결과
대기

해석: 이 테스트는 브라우저 안에서 짧게 반복 연산을 수행해 참고 점수를 보여주는 기능입니다. 전문 벤치마크가 아니며, 전원 모드·발열·백그라운드 작업·기기 설정에 따라 결과가 달라질 수 있습니다.

Manual input

정확한 부품명은 직접 입력하세요

웹에서는 정확한 CPU 모델명이나 저장장치 모델을 자동으로 읽을 수 없습니다. 옆의 도움말 버튼을 눌러 확인 위치를 보고 직접 입력하면 용도별 의미를 더 잘 해석할 수 있습니다.

직접 입력한 사양은 브라우저 화면에서만 해석하며, 한눈체크 자체 DB에 저장하지 않습니다.

Meaning

자동 확인 결과의 의미

Manual result

직접 입력 사양 해석

CPU, RAM, GPU, 저장장치 또는 사용 목적을 입력하면 이곳에 의미가 표시됩니다.

사용 방법

자동으로 볼 수 있는 정보와 직접 입력해야 하는 정보를 나눠 보여줍니다.

  1. 자동 정보 확인
    운영체제 추정, 논리 코어 수, 메모리 추정, GPU/WebGL, 화면 해상도를 확인합니다.
  2. CPU 간단 테스트
    짧은 반복 연산으로 현재 기기의 참고 성능을 확인합니다.
  3. 부족한 항목 직접 입력
    정확한 CPU 모델명, RAM, GPU, 저장장치는 사용자가 직접 입력합니다.
  4. 의미 확인
    입력한 사양이 어떤 작업에 유리한지 참고 설명을 확인합니다.

한눈체크의 방향

단순 숫자 나열이 아니라, 그 결과가 어떤 의미인지 설명합니다.

CPU 코어 수, 메모리, GPU, 화면 해상도는 숫자만 보면 어렵습니다. 한눈체크는 확인 가능한 정보와 직접 입력한 정보를 합쳐 문서 작업, 영상 편집, 게임, 개발 작업 관점에서 어떻게 받아들이면 좋은지 설명합니다.

FAQ

컴퓨터 사양 확인 자주 묻는 질문

왜 정확한 CPU 모델명이 자동으로 안 나오나요?

웹브라우저는 보안과 개인정보 보호 때문에 CPU 모델명, 저장장치 모델명, 메인보드 정보 같은 하드웨어 세부 정보에 직접 접근할 수 없습니다.

CPU 간단 테스트는 믿을 만한가요?

전문 벤치마크가 아니라 참고용입니다. 배터리 모드, 발열, 백그라운드 작업, 절전 설정에 따라 결과가 달라질 수 있습니다.

메모리 추정값이 실제와 다를 수 있나요?

네. 일부 환경에서는 메모리 정보가 제공되지 않거나 대략적인 값으로 표시됩니다. 정확한 용량은 운영체제의 시스템 정보에서 확인하는 것이 좋습니다.

입력한 사양이 저장되나요?

저장하지 않습니다. 직접 입력한 값은 현재 페이지에서 해석 결과를 보여주는 데만 사용됩니다.
