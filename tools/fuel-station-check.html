주유소 확인 가이드

최저가 주유소 확인은 이렇게 활용하세요.

주유소 가격은 유종, 지역, 갱신 시점에 따라 달라집니다. 오피넷 실제 데이터 기준으로 조회하며, 입력한 주유량과 연비를 바탕으로 예상 주유비와 참고 절약액을 계산합니다.

유종 선택

휘발유, 경유, 고급휘발유, LPG 중 실제 차량에 맞는 유종을 선택해야 가격 비교가 의미 있습니다.

가격과 거리 함께 보기

가장 저렴한 주유소가 멀리 있다면 이동 비용이 더 커질 수 있으므로 가격, 거리, 예상 절약액을 함께 봐야 합니다.

주유량 기준 계산

30L, 50L처럼 실제 주유량을 입력하면 최저가와 평균가 기준 예상 주유비 차이를 바로 확인할 수 있습니다.

부가서비스 확인

세차장, 경정비, 편의점 여부는 향후 오피넷 상세정보와 함께 표시할 예정입니다.

데이터 한계

가격은 제공기관 데이터 기준이며 현장 가격과 다를 수 있습니다. 방문 전 현장 가격을 확인하는 것이 좋습니다.

최저가 주유소 확인 FAQ

현재 가격은 실제 오피넷 가격인가요?

네. 주유소 확인하기를 누르면 한국석유공사 오피넷 API 기준 지역별 최저가 또는 반경 내 주유소 가격을 조회합니다. 예상 주유비는 조회된 리터당 가격에 사용자가 입력한 주유량을 곱해 계산합니다.

오피넷 API 키가 없으면 어떻게 되나요?

Cloudflare Pages Production 환경변수에 OPINET_API_KEY가 없으면 실제 가격 조회가 되지 않고 설정 안내가 표시됩니다.

주유비 절약액은 어떻게 계산하나요?

선택한 주유량 기준으로 전국 평균가와 최저가 주유소의 금액 차이를 계산합니다. 현재 위치 기반 조회에서는 주유소 왕복 이동거리와 연비를 반영한 참고 절약액도 함께 표시합니다.

지도 이동마다 API를 호출하나요?

아닙니다. API 호출량 절약을 위해 검색 버튼을 눌렀을 때만 조회하고, 상세정보는 카드 선택 시 호출하는 방식으로 설계합니다.

카카오맵 버튼은 무엇을 여나요?

주유소명과 주소를 기준으로 카카오맵 검색 화면을 엽니다. 정확한 장소 링크는 향후 데이터 상태에 따라 보강할 수 있습니다.
