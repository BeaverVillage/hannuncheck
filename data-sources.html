데이터 출처 안내

한눈체크는 국세청 사업자등록정보 API와 공정거래위원회 통신판매사업자 등록상세 API를 Cloudflare Pages Functions에서 호출하는 구조로 구현했습니다. API 키는 브라우저에 노출하지 않고 Cloudflare 환경변수로 관리합니다.

연동 데이터

  • 국세청 사업자등록정보 진위확인 및 상태조회 서비스
  • 공정거래위원회 통신판매사업자 등록상세 제공 서비스
  • 사용자가 입력한 쇼핑몰 하단 정보와 공식 등록정보 비교 결과

표시 원칙

조회 결과 화면에는 조회 시각, 등록정보 확인 결과, 입력정보 비교 결과, 거래 안전을 보장하지 않는다는 안내를 함께 표시합니다. 데이터 제공기관의 반영 시점에 따라 실제 상태와 차이가 있을 수 있습니다.
